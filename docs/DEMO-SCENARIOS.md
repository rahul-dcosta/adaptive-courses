## Quality Checks

For each generated course, verify:
- [ ] Course title is specific and relevant
- [ ] 3-5 modules (not too few, not too many)
- [ ] Each module has 2-4 lessons
- [ ] Lessons have meaningful content (200-300 words)
- [ ] Quizzes test understanding (not trivial)
- [ ] Next steps are actionable
- [ ] Estimated time matches timeline selection
- [ ] Tone matches goal (casual for "sound smart", detailed for "understand")

## Persona Variations

### The "Impress Them" User
- Wants buzzwords and frameworks
- Needs to sound smart in 24 hours
- Values tactical tips over deep understanding

### The "Survive This" User
- Under pressure (interview, meeting tomorrow)
- Needs just enough to not embarrass themselves
- Wants cheat sheets and quick references

### The "Actually Learn" User
- Has time and curiosity
- Wants comprehensive understanding
- Values depth over speed

### The "Career Switcher"
- Needs foundational knowledge fast
- Wants practical applications
- Values real-world examples
