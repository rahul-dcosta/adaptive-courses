# Adaptive Courses

AI-powered personalized course generator. Learn anything in minutes.

**Product:** Generate custom learning courses from minimal input.
**Pricing:** $5/course
**Stack:** Next.js + Supabase + Claude API + Stripe

## Quick Start
```bash
cd app
npm install
npm run dev
```

## Structure
- `app/` - Next.js application (Vercel deploys this)
- `docs/` - Product requirements & decisions
- `marketing/` - Copy, social posts, launch plan
- `analytics/` - Metrics & KPIs
- `sales/` - Outreach templates, pricing
- `decks/` - Pitch decks & presentations

## Tech Stack
- Frontend: Next.js 14 (App Router)
- Database: Supabase (auth + postgres)
- AI: Claude API (Sonnet for generation, Haiku for quizzes)
- Payments: Stripe
- Deploy: Vercel

Built by one person + AI team.
