# Adaptive Courses - Key Metrics

## North Star Metric
**Courses Generated Per Week**

## Core KPIs

### Acquisition
- Landing page visitors
- Click-through rate on "Generate Course" CTA
- Traffic sources (organic, social, PH, referral)

### Activation
- Users who start course creation flow
- Users who complete refinement questions
- Time to complete input → payment

### Revenue
- Payment conversion rate (started flow → paid)
- Revenue per week
- Average revenue per user (ARPU) = $5 (for now)
- Stripe fees vs net revenue

### Product
- Course generation success rate (should be ~100%)
- Average generation time (target: <60 seconds)
- Course quality (future: user ratings)

### Retention (Future)
- Users who generate 2+ courses
- Return visitor rate

## Dashboard (To Build)
- Real-time: Courses generated today
- Daily: Revenue, conversions, traffic
- Weekly: Trends, sources, top topics

## Tools
- Analytics: Vercel Analytics + custom Supabase queries
- Payments: Stripe Dashboard
- Errors: Sentry (if needed)
